# Tik_Book

Tik_Book 是一个旨在减轻阅读压力的项目，通过提供电子书的随机短片段，使阅读过程像浏览短视频一样轻松有趣。如果你喜欢某个片段，可以选择继续阅读更多。

## 功能特点

- **轻松阅读**：无需从头到尾阅读整本书，随机获取片段，享受阅读的乐趣。
- **发现新书籍**：通过随机片段，发现可能感兴趣的新书籍。
- **简单操作**：用户界面简洁，点击即可获取新的书籍片段。

## 开始使用

### 安装要求

确保您的系统已安装以下软件：

- Python 3.6+
- Pip（Python包管理器）

### 安装和运行

1. 克隆仓库：

```bash
git clone 
```
安装依赖：
```bash
cd tik_book
pip install -r requirements.txt
```

运行应用：

```bash 
python -m src.main
```
